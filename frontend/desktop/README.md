# Desktop Frontend

This repo now includes a native Windows desktop wrapper for the new work assistant.

Current implementation:

- Python desktop launcher
- local FastAPI backend started in-process
- native window via `pywebview`
- same UI and API contract as the web app

Entry point:

- [sikha_desktop.py](/e:/My%20project%20with%20git/AI-Asistant/sikha_desktop.py)

Build:

- [scripts/build_windows.ps1](/e:/My%20project%20with%20git/AI-Asistant/scripts/build_windows.ps1)

Launcher:

- [run_sikha.bat](/e:/My%20project%20with%20git/AI-Asistant/run_sikha.bat)
